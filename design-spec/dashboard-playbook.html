Especificaciones

Layout

  • ·240px sidebar + flex main + 340px rail
  • ·Screen body: height 640px overflow hidden
  • ·Main: flex column, inner scrollable

Componentes

  • ·Setup cards 4-col grid con sparkline SVG
  • ·Stats: WIN%, AVG R, CUM R por setup
  • ·Setup × Sesión cross-table (NY AM/PM/London/Asia)
  • ·A+ Checklist progress bars de cumplimiento
  • ·Toggle Pausados/Descartados
  • ·Tab switcher (Playbook activo)

Comportamiento

  • ·Cards color-coded por performance: verde=positivo, amarillo=neutral, rojo=negativo
  • ·Cross-table heat-map de fondo por win rate
  • ·A+ checklist fill-warn (<70%) marca en amarillo
  • ·Toggle Pausados/Descartados filtra el grid de setups
