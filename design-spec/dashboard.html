Especificaciones

Layout

  • 240px sidebar fija
  • flex:1 área main
  • 280px right rail
  • Shell full-height dark
  • 1440px canvas referencia

Componentes

  • Tab switcher (4 tabs)
  • KPI grid — 4 cards
  • Equity/Balance chart comparativo
  • Heatmap calendario
  • Data table trades
  • Direction toggle LONG/SHORT
  • Progress bars (ok/warn/crit)
  • Donut discipline score

Comportamiento

  • Tab switcher JS sin recarga
  • Balance+Equity: ambas líneas visibles
  • Heatmap coloreado por P&L
  • Right rail siempre visible
  • SEM badge actualiza cada lunes

Resumen de Funcionalidad

/dashboard

¿Qué hace?

  • → Vista central del estado de trading: cuenta, performance y disciplina
  • → Tab Portfolio: balance/equity, heatmap calendario, últimos trades
  • → Tab Operador: equity curve, métricas de sesión, rachas win/loss
  • → Tab Disciplina: score de disciplina, cumplimiento de reglas, progreso
  • → Tab Playbook: performance por setup, win rate y avg R comparados
  • → Right rail siempre visible: top setups + formulario nuevo trade

Tabs

  • Portfolio — balance total, equity, heatmap de P&L mensual, lista de trades recientes
  • Operador — equity curve Balance vs Equity, stats por sesión (NY AM/PM, London, Asia), rachas
  • Disciplina — score global 0-100, cumplimiento de reglas, violaciones, progreso semanal
  • Playbook — tabla de setups ordenada por avg R, win rate por setup, trades por setup
