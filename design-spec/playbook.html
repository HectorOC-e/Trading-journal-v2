Especificaciones

Layout

  • ·240px sidebar + 460px setup list + flex detail + 340px rail
  • ·Screen body height 560px, overflow hidden
  • ·Setup list: scrollable, fixed 460px width
  • ·Detail panel: flex:1, scrollable
  • ·Top bar inside main, sticky, border-bottom

Componentes

  • ·Setup cards con avatar cuadrado 32px, border-radius 8px
  • ·Badges ACTIVE (#1a2e1a / #4caf7a) y PAUSED (#2a2a1a / #c8a840)
  • ·Selected card: border-left 3px accent + accent bg 7%
  • ·Stats grid 3×2 con border-radius overflow
  • ·Checklists 2-col con check-box SVG checkmark
  • ·3 image placeholder boxes flex-row 33% each
  • ·Rail: Top performers leaderboard + Nuevo Trade form

Comportamiento

  • ·Click setup card → carga detalle en panel derecho
  • ·Valores financieros en JetBrains Mono siempre
  • ·Win color (oklch green) / Loss color (oklch red)
  • ·"+" abre modal de nuevo setup
  • ·"Ver trades" filtra tabla Trades por este setup
  • ·"Comparar" activa modo multi-select para comparar setups

Resumen de Funcionalidad

/playbook

¿Qué hace esta pantalla?

  • → Muestra el catálogo completo de setups operacionales del trader
  • → Permite ver performance histórico por setup (win rate, avg R, net P&L)
  • → Muestra el checklist de condiciones A+ y estándar por setup
  • → Exhibe screenshots de referencia para cada setup
  • → Permite pausar setups y comparar performance entre varios setups

Flujos de usuario

  • 1. Trader navega a /playbook → ve lista de setups ordenados por performance
  • 2. Click en setup → panel derecho muestra detalle completo
  • 3. Click "Ver trades" → navega a /trades filtrado por ese setup
  • 4. "+" → abre modal para crear nuevo setup
  • 5. "Comparar" → activa modo comparación entre 2+ setups
