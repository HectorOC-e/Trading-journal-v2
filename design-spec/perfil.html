Especificaciones

Layout · Componentes · Comportamiento

Layout

  • 240px sidebar fija
  • flex:1 main (2 cols: 1fr 1fr)
  • 280px right rail
  • Padding main: 28px 32px

Componentes

  • Avatar circle (64px, initials)
  • Form inputs (bg: --chip)
  • Toggle dark mode
  • Plan card + upgrade CTA
  • Security section
  • Guardar cambios button (accent)

Comportamiento

  • Form con validación en submit
  • Toggle dark mode → cambia clase body
  • Upgrade plan → modal/página de planes
  • Cambiar contraseña → modal
  • 2FA → flujo de configuración
  • Sidebar footer: CTA review semanal

Resumen de Funcionalidad

/perfil

¿Qué hace?

  • → Gestión de la cuenta del usuario
  • → Edición de información personal (nombre, timezone, idioma)
  • → Visualización y gestión del plan de suscripción
  • → Preferencias de la aplicación (tema, notificaciones)
  • → Seguridad: cambio de contraseña y activación de 2FA

Flujos de usuario

  • 1. Editar nombre/timezone → "Guardar cambios" → POST a API
  • 2. "Actualizar plan" → redirect a página de precios
  • 3. "Cambiar contraseña" → modal con campo actual + nuevo + confirmar
  • 4. "Activar 2FA" → flujo de QR code
  • 5. Toggle dark mode → cambia tema en tiempo real + persiste en localStorage
